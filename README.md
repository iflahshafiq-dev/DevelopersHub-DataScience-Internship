# DevelopersHub Data Science & Analytics Internship

This repository contains the tasks completed during the Data Science & Analytics Internship at DevelopersHub Corporation.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Tasks Completed

### Task 1: Iris Dataset Exploration
- Data loading and inspection
- Scatter plot, Histogram, Box plot
- Basic data visualization

### Task 2: Credit Risk Prediction
- Data cleaning
- Exploratory Data Analysis
- Logistic Regression model
- Model evaluation using Accuracy and Confusion Matrix

### Task 3: Customer Churn Prediction
- Data preprocessing
- Categorical encoding
- Classification modeling
- Feature importance analysis

### Task 4: Insurance Claim Amount Prediction
- Linear Regression model
- Visualization of Age, BMI, and Smoking impact
- Model evaluation using MAE and RMSE

### Task 5: Personal Loan Acceptance Prediction
- Customer data exploration
- Logistic Regression model
- Business insights from customer data

## Conclusion
These tasks demonstrate practical applications of data analysis, visualization, and machine learning techniques using Python.
